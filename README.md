# file_uploader_ui
File Uploader 화면


Front End
    - 작업 환경: Visual Studio Code
    - 개발 언어: Typescript
    - 주요 사용 라이브러리: React, Mobx, Material-ui
    - 개발 환경
        - node 버전: 18.12.0
        - back end api 주소
            - localhost:8080 (axios 코드 참조)
    - 실행 환경
        - node 버전을 맞춘 후, 'fileuploader-ui' 디렉토리에서 다음 명령어로 실행 가능
            1. yarn
            2. yarn start
